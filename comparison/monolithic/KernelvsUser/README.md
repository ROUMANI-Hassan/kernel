# Kernel space compared with user space

This section compares programs in user space with code loaded into kernel
space. The examples study communication and isolation between the two areas.

## Examples

| Folder | Comparison |
|---|---|
| `device_communication/` | A user application reads data from a kernel module through a device |
| `stackoverflow/` | The same recursive stack exhaustion in a process and a kernel module |

Open the README inside an example for its exact commands and expected results.
